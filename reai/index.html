🧑‍🏫 lecturer flow
Create a test (dept)
Set department · question pool · presentation rules
1️⃣ Select department & course
2️⃣ Build question bank (100+ questions)
3️⃣ Set “present 30” randomly from pool
4️⃣ Assign 30 marks (1 mark each)
5️⃣ Publish · schedule · invigilator
